### 1. `<rss>`
Defines the RSS version used in the document.

### 2. `<channel>`
Contains all the main metadata of the feed:
- **Title**: General headline of the RSS feed
- **Link**: Main reference website (Netflix Spain)
- **Description**: Brief explanation of the feedâ€™s theme
- **Language**: Spanish (`es`)

### 3. `<item>`
Each `<item>` represents a Netflix series and includes:
- **Title**: Name of the series
- **GUID**: Unique identifier and link to the Netflix page
- **Description**: Summary of the series content and its thematic focus
